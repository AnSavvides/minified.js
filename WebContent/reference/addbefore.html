.addBefore()

Inserts the given text or element(s) as sibling in front of each element of this list.

Syntax

MINI(selector).addBefore(text)
MINI(selector).addBefore(callbackFunction)
MINI(selector).addBefore(elementContent)

Parameters

text
a text to insert as text node in front of the list elements
callbackFunction
a factory function(element, index) that will be invoked for each list element to determine its content. The function can return either a string for a text node, an HTML element or a list containing strings and/or HTML elements. As parameters it receives the current list element and its 0-based index.
elementContent
content to insert in front of the first element of the list. The content can be a string for a text node, an HTML element or a list containing strings and/or HTML elements. Note that if you use a HTML node or an array here, it will only be added once to this list's first match.
return value
the current list

Description

Inserts the given text or element(s) as sibling in front of each element of this list. If a string has been given, it will be added as text node. If you pass an element or a list, it will be inserted only in front of the first element of the list. In order to add elements to several list items, you need to pass a factory function(element, index) thats creates new instances for each item. The function will be invoked for each element and may return either a string for a text node, a simple HTML element or a list containing both.

Example

Using the following HTML: 
<div>
  
Here is some text

</div>
addBefore() adds text in front of the selected list items. 
$('#mainText').addBefore('COMMENT');
This results in: 
<div>
  COMMENT
  
Here is some text

</div>

Example

You can also pass an element, but note that a node can only be added once to the first list item, even if the list has more than one item: 
$('#mainText').addBefore(EE('span', {'className': 'important'}, 'WARNING'));
With the previous example's HTML, this would create this: 
<div>
  <span class="important">WARNING</span>
  
Here is some text

</div>

Example

Lists of elements and nodes are possible as well. Like nodes, it will only be added to the first list item. 
$('#status').addBefore([EE('hr'), 'WARNING']);

Example

If you need to add your list to more than one list item, you must add a factory function that re-creates the list for every instance: 
$('.textSnippets').addBefore(function(e, index) { return [EE('hr'), 'WARNING']; });
