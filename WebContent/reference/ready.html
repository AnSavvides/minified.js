MINI.ready()

Registers a handler to be called as soon as the HTML has been fully loaded (but not necessarily images and other elements).

Syntax

MINI.ready(handler)

Parameters

handler
the function to be called when the HTML is ready

Description

Registers a handler to be called as soon as the HTML has been fully loaded (but not necessarily images and other elements). On older browsers, it is the same as 'window.onload'.

Example

Register an handler that sets some text in an element: 
MINI.ready(function() {
  $$('#someElement').innerHTML = 'ready() called';
});
