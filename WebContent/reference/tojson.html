$.toJSON()

Converts the given value into a JSON string.

Syntax

$.toJSON(value)

Parameters

value
the value (map-like object, array, string, number, boolean or null)
(return value)
the JSON string

Description

Converts the given value into a JSON string. The value may be a map-like object, an array, a string, number, boolean or null. If you build Minified without Internet Explorer compatibility, this is just an alias for JSON.stringify.

The following types are supported by the built-in implementation:

  • Objects (direct properties will be serialized)
  • Arrays
  • Strings
  • Numbers
  • Boolean
  • null
Any other types in your JSON tree, especially Dates, should be converted into Strings by you.

Example

Convert an object into a JSON object: 

var myObj = {name: 'Fruits', roles: ['apple', 'banana', 'orange']};
var jsonString = $.toJSON(myObj);
