.raw

Returns the creation object of this list, either an Array, a NodeList or another MINI list.

Syntax

raw

Description

Returns the creation object of this list, either an Array, a NodeList or another MINI list. This is mostly useful after calling filter(), as only then it is guaranteed to be an Array.

Example

$($('.myElement').filter().raw.slice(1, 3)).addClass('secondOrThird');
